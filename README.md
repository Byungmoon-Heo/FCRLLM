# FCRLLM
Aligning LLM with Collaborative Filtering for Long-tailed Sequential Recommendation

<br/>

## Environment Setting:

- numpy==1.24.4
- pandas==2.2.2
- scikit-learn==1.5.1
- scipy==1.14.0
- torch==2.5.0
- tqdm==4.66.5
- joblib==1.4.2
<br>
